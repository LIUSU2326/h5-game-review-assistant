# v1.4 证据复核增强记录

## v1.4.0-alpha.1

第一项增强是“导出证据包”。

用途：

- 将单款游戏的本地证据目录压缩成 zip。
- 方便把截图、报告、AI 结果、payload、复核记录和运行记录发给别人复核。
- 仍然遵守“视频本地保存、截图上传飞书可选”的原则，导出动作不会上传任何文件。

界面入口：

```text
证据复核 → 右侧选中游戏面板 → 导出证据包
```

导出位置：

```text
数据目录\outputs\evidence-packages\<game_id>-evidence-<timestamp>.zip
```

已验证：

- `cow-saver` 可成功导出证据包。
- 导出文件大小约 1.94 MB。
- 界面中可看到“导出证据包”按钮。
- `npm run app:check` 仍然通过。

## v1.4.0-alpha.2

第二项增强是“截图放大预览”。

用途：

- 在证据复核面板直接点击截图缩略图放大查看。
- 保留“打开原图”入口，方便继续用系统图片查看器或浏览器检查细节。
- 支持按 Esc 关闭预览层。

已验证：

- 点击 `cow-saver` 截图缩略图可打开预览层。
- 预览层显示原图路径和图片内容。
- Esc 可关闭预览层。
- 1600x1000 桌面窗口无页面级溢出。
