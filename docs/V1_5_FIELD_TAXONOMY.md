# v1.5 字段和标签库增强记录

## v1.5.0-alpha.1

第一项增强是“模板版本审计”。

用途：

- 让使用者知道当前飞书多维表格是否和工具字段库一致。
- 把本地字段映射、飞书远端字段和标签库同步状态合成一个可复制摘要。
- 给每次字段模板状态生成稳定指纹，方便排查“字段是不是变过”。

界面入口：

```text
配置中心 → 字段与标签库 → Template audit
```

包含信息：

- 模板版本号，例如 `tpl-36-63a1ddd0`。
- 本地字段映射 SHA256。
- 飞书远端字段结构 SHA256。
- 标签库结构 SHA256。
- 预期字段数、飞书字段数、缺失字段、类型风险、标签选项数。
- 下一步处理建议。

已验证：

- 当前模板版本：`tpl-36-63a1ddd0`。
- 预期字段：36。
- 飞书字段：37。
- 缺失字段：0。
- 类型风险：0。
- 标签选项：49。
- 配置中心 1600x1000 桌面窗口无页面级溢出。
- `npm run app:check` 通过。
