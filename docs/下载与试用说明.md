# H5 游戏评测助手下载与试用说明

## 下载

当前推荐试用包：

```text
downloads/H5游戏评测助手-v1.3.0.zip
```

预览新版 AI 自动试玩增强：

```text
downloads/H5游戏评测助手-v1.6.0-alpha.1.zip
```

解压后运行：

```text
H5游戏评测助手 1.3.0.exe
```

## 试用前准备

需要准备：

- Gemini API Key。
- 飞书自建应用的 App ID 和 App Secret。
- 一个已经添加文档应用、并授权可编辑的飞书多维表格。

如果暂时不接飞书，也可以只生成本地报告和写入预览。

## 建议试用顺序

1. 打开工具。
2. 进入“配置中心”，填写 Gemini 和飞书配置。
3. 点击“快速检查”。
4. 进入“工作台”，导入 1 个 H5 游戏链接。
5. 先用 `poc_review` 短测档跑通。
6. 检查本地证据文件夹和飞书写入结果。
7. 再扩大到多款游戏或 `full_30min` 正式档。
