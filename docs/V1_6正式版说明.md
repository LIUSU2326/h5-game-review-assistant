# v1.6.0 正式版说明

## 定位

`v1.6.0` 是 H5 游戏评测助手的 AI 自动试玩增强正式版。

## 下载包

```text
downloads/H5游戏评测助手-v1.6.0.zip
```

解压后运行：

```text
H5游戏评测助手.exe
```

## 主要能力

- `adaptive_probe` 自动试玩策略。
- `ai_probe_alpha` 运行档位，用于验证 AI 自动试玩增强。
- 运行前预检：开始运行前提示耗时、策略、Gemini、飞书和批量风险。
- 自动试玩复盘：运行后汇总策略、动作数、档位数、动作类型和重跑建议。
- 本地 HTML 报告补充自动试玩复盘摘要。

## 使用建议

1. 新链接先跑 `poc_review`。
2. 对需要更强自动试玩探索的游戏，再跑 `ai_probe_alpha`。
3. 确认截图、视频、复盘和字段无明显问题后，再跑 `full_30min`。
4. 大批量运行前先使用“预演队列”检查任务范围。

## 边界

- 自动试玩本身不逐帧调用 Gemini，不会因为自动试玩明显增加 API 消耗。
- Gemini 消耗主要来自 AI 字段生成。
- 视频只保存本地。
- 截图上传飞书是可选功能。
- 小程序、App、真机或云真机将在后续版本规划。
